struct net_port * net_get_port_list ( int  host_id)

Retrieves a list of net_port structures for a given host ID.

This function retrieves a list of net_port structures that match the specified host ID, either as pipe_host_id or sock_host_id. The function modifies the global port list and creates a new list with the matching net_port structures.

Parameters
host_idThe ID of the host to search for.
Returns
A pointer to the first net_port structure in the list of matching ports, or NULL if no matching ports are found.
